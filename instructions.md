This project is a small **interactive web demo** that teaches **Prompt Engineering** 🧠.
It does **three main things**:

### 🌟 Shows why prompt engineering matters:

It explains that **good prompts** help people get **precise AI responses**, **reduce retries**, **lower token usage/cost**, and **work better** with AI tools 🛠️.

### 🧪 Demonstrates prompting techniques:

It includes **three techniques**:

* **Zero-shot** ⚪
* **Few-shot** 🔵
* **Structured prompting** 🏗️

### ✍️ Lets the user build a prompt:

The user enters a **task goal**, **audience**, and **constraints**. The app then **automatically generates** a better AI prompt based on the selected technique ✨.

* 📂 **Created the project folder**
The demo lives in: `create-a-demo-project-prompt-engineering`

* Main title: **Prompt Engineering** 🏷️

This file controls the **design**:

This file adds **interactivity**:
* Detects which technique button is selected ✅

* Updates the output **live** ⏱️
* **Copies** the generated prompt to clipboard 📋

This explains what the project is and **how to run it**.
* ✅ **Tested the project**
I opened it in the browser and verified:
* The page **loads correctly**

* The generated prompt **updates**

* 📦 **Created a zip file**
The final downloadable package is: `prompt-engineering-demo.zip`
