# Prompt Engineering Demo

A small static demo project for introducing prompt engineering concepts.

## What It Shows

- Why prompt engineering matters for anyone entering AI.
- How precise prompts improve AI responses.
- How better prompts can reduce retries, token usage, and cost.
- Zero-shot, few-shot, and structured prompting techniques.

## Run

Open `index.html` in a browser.

No build step or package install is required.
